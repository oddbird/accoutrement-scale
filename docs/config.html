$ratios (map)

scss
$ratios: () !default;

Description

Example

scss
$ratios: (
  'line-height': 1 / 3,
  'headings': 'linear',
  'golden': 1.61803399,

$sizes (map)

scss
$sizes: (
  'root': $_BROWSER-DEFAULT-FONT-SIZE,
  'text': 'root' ('convert-units': 'rem'),
  'rhythm': 'text' ('fifth': 1),

  'h1': 'text' ('linear': 4),
  'h2': 'text' ('linear': 3),
  'h3': 'text' ('linear': 2),
) !default;

Description

Defined a palette of common sizes to be used across your project, in the format "name: size" or "name: basis (ratio/function: value)". If your map includes a value for root, it will be used in relative-size unit conversions.

Map Properties

'root': 16px (Map)

Include a root font-size for your document, used for calculating relative sizes. This should match the size applied to your html element.

used by

@function size()

@function convert-units()
